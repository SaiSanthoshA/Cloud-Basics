# Cloud-Basics
It is a simple diagram to explain the concept of cloud working.
VPC: It is a Basic form fo cloud deployment where it known as Virtual Private Cloud used to work your personal data with cloud models like AWS. It can build it's own IP address
Subnets: It is used as subdivisions in the VPC to organize and secure the resources based on their roles. There are 2 types they are 1)Private 2)Public
1)Private Subnets: It is used to protect the data in the backend resources. No direct route to the internet.
2)Public Subnets: It is used to get the data available on the internet through internet gateway.

Internet Gateway: It is used to connect the internet and the VPC together.
Security Group: It is used to protect the data with EC2 instances that controls the traffics.
